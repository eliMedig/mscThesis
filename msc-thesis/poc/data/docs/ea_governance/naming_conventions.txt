Application Naming and Identification Conventions (sample)

Application ID (app_id)
- Format: APP-NNNN (uppercase 'APP', hyphen, four digits).
- The app_id is immutable for the life of the application and must be unique.

Application name (app_name)
- Human-readable, unique within the portfolio.
- Should not encode hosting or lifecycle (those are separate attributes).

Hosting attribute
- One of: on_prem, cloud, hybrid.
- 'hybrid' requires documentation of which components run where.
